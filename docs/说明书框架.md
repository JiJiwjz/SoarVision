## 1 作品概述
- 1.1 问题背景:琼州海峡常年大雾(能见度<500m),传统摄像头失效,威胁航行安全 → 需多模态智能感知。〔C3.md 原文〕
- 1.2 任务定义:海上目标(船舶/小船/浮标)识别,聚焦雾/低光/噪声等限定条件。3 类。
- 1.3 创新点概览(对应赛题三轴):**多模态融合**(雷达-相机 query 级软关联)/ **抗干扰**(物理退化鲁棒训练 D1)/ **轻量化**(RF-DETR nano + ONNX/TRT 可部署)。

## 2 系统总体方案
- 2.1 总体框图:输入(RGB + 毫米波雷达[+仿真深度])→ 退化鲁棒检测主干 → 雷达融合 → 输出(框/类/置信)。〔补一张架构图〕
- 2.2 模态与传感器:RGB 相机、TI IWR6843 毫米波雷达(77GHz)、Jetson Orin Nano。
- 2.3 数据流与推理流程。

## 3 数据集
- 3.1 检测数据:SeaShips + SMD-VIS(3 类,70/15/15 划分)。
- 3.2 雷达-相机数据:WaterScenes(77GHz 4D 雷达,与我方同频段;7类→映射3类)。〔A1 已落地+对齐验证〕
- 3.3 退化合成管线:大气散射雾 + 低光 + 传感器噪声(论文背书)。〔scripts/degrade.py〕

## 4 核心算法(重点章节)
- 4.1 检测主干 **RF-DETR**:为何选(2026 SOTA 实时 DETR,DINOv2+NMS-free),高分辨率@896 攻小目标。
- 4.2 **退化鲁棒训练 D1**:大气散射模型 I=J·t+A(1−t);联合清晰+退化训练;论文背书(Sakaridis/Hendrycks/Foi/Brooks)。〔结果:浓雾+5.5,见第6章〕
- 4.3 **多模态雷达融合 RGQ**:REVP 表示 → 雷达编码器 → decoder query 级 cross-attention 软关联(TransCAR 路线)+ 零初始化门控("只增不减")。〔radar/fusion.py + integrate.py〕
- 4.4 其他:抗海杂波先验、小目标策略。

## 5 实现与部署
- 5.1 训练:RF-DETR + D1 + 自定义融合训练循环(复用 RF-DETR criterion/matcher)。
- 5.2 导出与部署:PyTorch→ONNX(已验证可转)→ Jetson 上 TensorRT FP16。〔X1:算子全可转〕
- 5.3 实时性:5090 占位数 + Jetson 目标(国赛补)。
