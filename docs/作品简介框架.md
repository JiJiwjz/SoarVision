## 作品名称
**翱翔智眸(SoarVision)** — 面向雾天海上目标的多模态智能感知系统。(一句话定位)

## 一、背景与问题(1 段)
琼州海峡常年大雾、能见度常 <500m,传统光学摄像头失效,海上船舶/浮标/障碍物漏检,严重威胁航行安全。亟需在恶劣天气下仍可靠的智能目标感知。

## 二、方案概述(1 段)
以最新实时检测 Transformer **RF-DETR** 为主干,叠加**物理退化鲁棒训练(D1)**抵御雾/低光/噪声,并引入**毫米波雷达-相机 query 级融合(RGQ)**,在光学失效时由雷达补盲;全链可导出 ONNX→TensorRT 部署于 Jetson。

## 三、主要创新点(3–4 条 bullet)
- **多模态融合**:雷达-相机 query 级 cross-attention 软关联 + 零初始化门控("只增不减",雾/夜放大雷达)。
- **抗干扰**:基于大气散射物理模型的退化鲁棒训练(论文背书),浓雾恢复显著。
- **轻量可部署**:RF-DETR nano + ONNX/TensorRT,面向 Jetson 实时。
- **工程严谨**:论文背书 + 累加消融 + git commit 全程留痕。

## 四、关键结果(几个亮眼数字)
- 浓雾 mAP@50:95 **+5.5**(small)/ **+7.0**(nano),且晴天不降反升。
- 抗雾稳定性 ×5(晴→浓雾掉幅 −0.057 → −0.011)。
- 退化下小目标 recall 成倍提升;ONNX→TRT 已验证可转换(算子无障碍)。
- 〔多模态融合消融数字:A4 完成后填〕

## 五、应用价值(1 段)
琼州海峡及类似强雾航道的无人艇/航行安全感知;可扩展至港口监管、海上搜救等。
